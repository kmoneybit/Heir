# Thaniablaq Hair - Local Server

This project is a static storefront with a minimal Express backend and SQLite DB for product management.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run server

```bash
npm start
```

3. Open site

- Frontend: http://localhost:3000/
- Admin: http://localhost:3000/admin

Notes:

- The server creates `data.db` on first run and seeds a couple products.
- This is a development/demo setup. For production deploy, secure the admin and use environment-configured DB.
